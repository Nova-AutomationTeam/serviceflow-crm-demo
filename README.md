# ServiceFlow CRM Demo

ServiceFlow CRM is a polished portfolio demo of a custom CRM system for small service businesses.

The project shows how businesses can manage leads, clients, deals, tasks, and revenue analytics in one clean dashboard.

Public visitors can explore read-only CRM pages without login. Django Admin is kept private and intended only for internal project owner access.

## Features

- CRM dashboard
- Lead management
- Client database
- Sales pipeline
- Task tracking
- Revenue analytics
- Realistic demo data
- Private Django Admin
- Read-only public demo pages

## Tech Stack

- Python
- Django
- SQLite
- Django Templates
- Tailwind CSS
- Chart.js
- Django Admin

## Portfolio Use Case

This project was built as a portfolio demo to showcase full-stack development, CRM architecture, business automation logic, dashboard UI, and secure public demo access.
